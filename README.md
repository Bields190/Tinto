## Cada Vinho, uma História
TINTO vem com a proposta de inovar a degustação de vinhos: trazendo uma plataforma na qual se torna possível catalogar, avaliar e favoritar vinhos já consumidos

Nosso objetivo é garantir que o usuário sempre se lembre de suas experiências bebendo um bom vinho.
