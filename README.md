# 🌴 Tinto - Webapp com Jungle Backend

Repositório criado como complemento da disciplina de Engenharia de Software I, ministrada pelo Professor Daricélio Soares na Universidade Federal do Acre (UFAC).

## 📋 Sobre o Projeto

Tinto é uma aplicação web moderna desenvolvida com **Flask** e **Jungle** (AWS) como backend. O projeto demonstra a implementação de uma arquitetura web completa, incluindo frontend responsivo, API REST e integração com serviços AWS.

## 🚀 Tecnologias Utilizadas

- **Python 3.12+** - Linguagem de programação
- **Flask 3.1.2** - Framework web para o servidor
- **Jungle 1.0.0** - Ferramenta de gerenciamento AWS
- **HTML5/CSS3** - Interface frontend
- **JavaScript** - Interatividade no cliente

## 📦 Instalação

1. Clone o repositório:
```bash
git clone https://github.com/Bields190/tinto.git
cd tinto
```

2. Instale as dependências:
```bash
pip install -r requirements.txt
```

## 🏃 Executando a Aplicação

Execute o servidor de desenvolvimento:

```bash
python app.py
```

A aplicação estará disponível em: `http://localhost:5000`
