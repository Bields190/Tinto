Sobre o Projeto Tinto

📚 Contexto Acadêmico

Este projeto foi desenvolvido como parte da disciplina de Engenharia de Software I, ministrada pelo Professor Daricélio Soares na Universidade Federal do Acre (UFAC).

🎯 Objetivo

Criar uma aplicação web moderna utilizando Jungle como backend, demonstrando conceitos de desenvolvimento web, arquitetura de software e integração com serviços em nuvem.

🛠️ Tecnologias Utilizadas

  • Flask - Framework web Python para o servidor
  • Jungle - Ferramenta de gerenciamento AWS para backend
  • HTML5/CSS3 - Interface frontend moderna
  • JavaScript - Interatividade no cliente

📋 Funcionalidades

  • Interface web responsiva
  • API REST para comunicação com backend
  • Integração com serviços AWS via Jungle
  • Arquitetura escalável e modular
